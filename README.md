This script here is to help you run your steam games that are install on a a NTFS or windows drive. This issue is known and listed here: 
https://github.com/ValveSoftware/Proton/wiki/Using-a-NTFS-disk-with-Linux-and-Windows#preventing-ntfs-read-errors

Thank you so much to bakudan-otaku for improving this script from my jankie personal use one to a more safe and easy to use script.

Usage:

./ntfs-fix.sh MEDIANAME STEAMID

Example: ./ntfs-fix.sh NVMe 1962700

Configures Subnautica 2 which is on NVMe

Follow the wiki for futher instructions.
